Facebook Profile Viewer

A simple web application that allows users to fetch their Facebook profile information using anAccess Token. Users can see their profile picture, name, and Facebook ID.
------------

Features

* Fetch profile picture, name, and Facebook ID from Facebook Graph API.

* Simple and responsive UI.

* Error handling for invalid or missing access tokens.

-----------

How to Use

1. Clone the repository:
git clone https://github.com/YOUR_GITHUB_USERNAME/facebook-profile-viewer.git

2. Openindex.htmlin your browser.

3. Paste your Facebook Access Token in the input field.

4. ClickFetch Profileto view your Facebook profile details.

------------

Technologies Used
   * HTML, CSS, JavaScript
   * Facebook Graph API
